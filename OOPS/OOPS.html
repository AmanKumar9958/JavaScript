OOPS is the programming paradigm (method) on the concept of object, which contains both data and functions

OOP helps in organizing code in a way that is more modular, reusable, and easier to maintain.

Parts of OOPS-

  • Constructor Function
  • Prototypes
  • Classes
  • Instance (this, new)

4 Pillar of OOPS-

  • Abstraction - Hiding the details
  • Encapsulation - Wrapping the data
  • Inheritance
  • Polymorphism
